# 기능 소개

CSS, JS를 이용하여 만든 디지털 시계로, AM과 PM이 화면에 표기되며 50분이 될 때마다 휴식 시간임을 알려 주는 알람이 울립니다.
애니메이션을 활용하여 흔들리는 효과를 추가했습니다.
